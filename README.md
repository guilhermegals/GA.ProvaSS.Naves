# Batalha de Naves
Projeto desenvolvido com a tecnologia Unity para a prova de seleção.

## Funcionalidades Desenvolvidas
* Movimentação das Naves
* Colisão
* Disparo
* Buraco negro
* Aceleração e gravidade

# Tecnologias Utilizados
* Unity
* C#
